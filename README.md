# QuickByte Predictor 🍔⚡

An AI-powered system to predict:
- Restaurant rush level (Low / Medium / High)
- Food preparation time
- Best restaurant for fastest service

## Features
- Smart recommendation system
- Restaurant comparison dashboard
- Food menu integration
- Real-time prediction graphs

## Tech Stack
- Python (Flask)
- HTML/CSS/JS
- Machine Learning (Random Forest)
- SQLite Database

## How to Run
pip install flask pandas scikit-learn matplotlib
python app.py
